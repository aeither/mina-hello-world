# Mina zkApp: Mina Hello World

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```


## Start a new project

```sh
zk p Name
```

## Prepare for deploy

```sh
zk config
```

## Deploy and interact

```sh
zk deploy berkeley
```

```sh
node build/src/interact.js berkeley
```

--

I can deploy only once. By default it is NOT updatable
